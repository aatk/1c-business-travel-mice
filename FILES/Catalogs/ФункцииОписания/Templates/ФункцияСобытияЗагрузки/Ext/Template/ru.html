Доступные параметры функций

При использовании событий при загрузке

РеальныйОбъект - Содержит объект в который произошла запись (Если переменная равен "Документы.Услуга.ПустаяСсылка", то записи не было)

НастройкаДляЗагрузки - Содержит текущую настройку загрузки

СтруктураУслуги - Содержит структуру загружаемого json

НовыйОбъект - Содержит структуру загружаемого объекта
