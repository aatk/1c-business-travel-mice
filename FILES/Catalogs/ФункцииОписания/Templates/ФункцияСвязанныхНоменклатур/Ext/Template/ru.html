Доступные параметры функций

При использовании обработчика установки цены связанной номенклатуры

Параметры - Содержит предопределенные значения

Объект- Услуга которая формирует связанные услуги
Связь - Объект Структура содержащая связь по которой требуется выполнить функцию (требуется обязательная проверка, на существование ключа)

Результат всегда требуется записывать в переменную "Результат". Пустой обработчик, во избежания ошибок, должен состоять из кода "Результат = Параметры;"
В "Параметрах" должно находиться хотябы одно значение с типом "Число", которое будет присвоено переменной.
