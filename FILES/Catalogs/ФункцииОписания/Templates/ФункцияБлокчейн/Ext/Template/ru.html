Доступные параметры функций

При использовании событий при загрузке

УслугиИсточника - Массив услуг которые будут проверяться в объекте

СтрокаСкриптов - Строка табличной части справочника "Ценности" содержит поля (Ссылка, ИдентификаторМетаданных, СкриптОтслеживания, РеквизитОбъектаСУслугой)

Источник - Объект который производит проведение

Отказ - При помещении в поле "Ложь", в документ будет отменено текущее действие(проведение)

РежимПроведения - Содержит режим проведения документа
