Доступные параметры функций

При использовании заполнения полей документа при загрузке

Параметры.ЗначениеJSON - Содержит значение поля json которое указано в настройке
Параметры.СтруктураJSON - Содержит всю структуру передаваемого json
Параметры.Услуга - Содержит сам объект документа Услуга, либо иного объекта в который проихводится загрузка данных
Параметры.ЗначениеЦикла - Содержит значение элемента при загрузке нескольких объектов по одному json
Параметры.ГлобальныйИндексУслуги - Содержит значение индекса загружаемого значения цикла
